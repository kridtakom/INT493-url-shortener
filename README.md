# INT493-url-shortener

# Prerequisites
- Docker
- Docker-compose

### 1. Clone Source Code
```sh
$ git clone https://github.com/kridtakom/INT493-url-shortener.git 
$ cd INT493-url-shortener 
$ chmod +x *.sh 
```

### 2. Run 
*** Don't forget change redis password in docker-compose file
```
$ ./docker-update&clear&run.sh
```
